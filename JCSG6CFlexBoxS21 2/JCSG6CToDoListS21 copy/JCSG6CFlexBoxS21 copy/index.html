To Do List Project

For this challenge, you will be creating a custom To-Do list. The purpose of this is to help you learn the basics of JavaScript while also learning basic logic and backend of web development. As a user, you should be able to add To-Do items via multiple form elements, complete those items on the list and remove them, and to toggle the theme of the app from light or dark mode.
